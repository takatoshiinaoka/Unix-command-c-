# unix-command-c
UnixコマンドをC言語で書くリポジトリ
## 使い方
- gccをインストール
  ```
  brew install gcc
  ```
- インストールの確認
  ```
  brew list | grep gcc
  ```
- ディレクトリ移動
  ```
  cd ls
  ```
- コンパイル
  ```
  gcc ls.c
  ```
- 実行
  ```
  ./a.out  
  ```
